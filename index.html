Snake Game with Phaser

Built with the latest Phaser framework, this snake game is a fun and interactive way to pass the time.

How to Play

Use the arrow keys to control the snake, eat food pellets to grow longer and avoid colliding with the walls or the snake's tail.

Game Features

  • Phaser framework
  • Easy to play
  • Increasing difficulty

Game Stats

Play the game to unlock your own personal stats, including your highest score and longest play time.
